# Setup Licensing

## Summary
Add dual-licensing files to the repository based on the licensing spec.

## Scope
- Add `LICENSE` file (GPLv3 full text)
- Add `LICENSE-COMMERCIAL.md` (commercial license description + contact info)
- Add license section to `README.md`

## Out of Scope
- CLA setup (separate change, needed before first external PR)
- Formal commercial license contract (only needed when a company inquires)

## References
- Spec: `openspec/specs/licensing/spec.md`
