# Init Tauri Project — Design

## Project Structure

```
Spool/
├── CLAUDE.md
├── .gitignore
├── openspec/              ← existing
├── src-tauri/             ← Rust backend (generated by Tauri)
│   ├── Cargo.toml
│   ├── src/
│   │   └── main.rs
│   ├── tauri.conf.json
│   └── icons/
├── src/                   ← Svelte frontend
│   ├── App.svelte
│   ├── main.js
│   └── styles.css
├── package.json
├── vite.config.js
└── index.html
```

## Approach
Use `create-tauri-app` CLI with Svelte template to generate the scaffold, then add project-specific dependencies.

## Dependencies

### Rust (Cargo.toml)
- `tauri` — framework
- `image` — standard image decoding
- No `libraw-rs` yet (add when RAW support is implemented)

### Frontend (package.json)
- `svelte` — UI framework
- `leaflet` — map library (add dependency now, integrate later)

## Notes
- Skip `libraw-rs` in initial scaffold to avoid C build complexity before it's needed
- Leaflet added as dependency but not wired up yet
