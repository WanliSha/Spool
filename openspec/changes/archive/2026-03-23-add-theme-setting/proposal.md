# Add Theme Setting

## Summary
Add a light/dark/system theme selector to the settings page, allowing users to override the OS default.

## Scope
- Add theme option to settings: Light / Dark / System (default)
- Persist theme choice in settings.json
- Apply theme on app startup
- Update CSS to use theme class instead of `prefers-color-scheme` media query

## Out of Scope
- Custom color themes
- Per-window theme
