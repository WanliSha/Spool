# Add File Import

## Summary
Implement drag-and-drop and file picker import for photos and folders.

## Scope
- Drag and drop files into the app
- Drag and drop folder (top-level only by default)
- File picker dialog (files and folder)
- Settings toggle: recursive subfolder loading (off by default)
- Display loaded photos as a flat list with source path indication
- Filter supported image formats on import

## Out of Scope
- Photo grid/list UI design (separate change)
- Thumbnail generation (separate change)
